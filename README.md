# Lira TV

一个功能丰富、视觉效果精美的个人网站，集成了灵动岛、Live2D 模型、音乐搜索和播放等多种功能，为用户提供沉浸式的浏览体验。

---

## 功能特点

### 1. 灵动岛 (Dynamic Island)

- 模拟 iOS 灵动岛的交互效果，提供现代化的用户界面
- 支持音乐搜索和播放控制，集成音乐功能
- 欢迎语显示与律动条动画，增强用户体验
- 流畅的展开/收起动画效果，提升视觉质感

### 2. Live2D 模型

- 多种角色可选：Haru、Hiyori、Mao、Mark、Natori、Rice、Wanko
- 生动的动画效果和表情变化，增加页面活力
- 与用户交互的响应效果，提供沉浸式体验

### 3. 音乐功能

- 音乐搜索和播放，支持多种音乐源
- 歌词显示，同步展示歌曲歌词
- 可拖动的音乐图标，自定义界面布局
- 收藏歌曲功能，方便用户管理喜欢的音乐

### 4. 视觉效果

- 音频可视化效果，随音乐节奏变化
- 打字机效果，增强文字展示的趣味性
- 点击特效，提供反馈和视觉刺激
- 流畅的动画过渡，提升页面质感

### 5. 聚合搜索功能

- 多数据源聚合搜索，提供全面的影视资源
- 横向滚动布局，优化空间利用
- 鼠标滚轮控制，提升操作便捷性
- 详细的剧集信息展示，包含完整的影视资料
- 支持多种API源，确保资源的丰富性

## 技术栈

- **HTML5** - 提供语义化标记，构建网站结构
- **CSS3** - 实现响应式设计和视觉效果
- **JavaScript (ES6+)** - 实现交互功能和动画效果
- **Live2D SDK** - 集成交互式2D模型
- **Font Awesome 图标** - 提供丰富的图标资源
- **Fetch API** - 处理网络请求，实现数据交互
- **CORS 代理** - 解决跨域问题，确保API正常调用

## 快速开始

### 方法一：使用临时服务器

1. 确保你的系统已安装 Python 3.6 或更高版本
2. 在项目根目录运行以下命令：
   ```bash
   python "临时网页服务器.py"
   ```
3. 打开浏览器访问：`http://localhost:8085`
4. 临时服务器已配置 CORS 代理，确保聚合搜索功能正常工作

### 方法二：使用 Python 内置服务器

1. 确保你的系统已安装 Python 3.6 或更高版本
2. 在项目根目录运行以下命令：
   ```bash
   python -m http.server 8085
   ```
3. 打开浏览器访问：`http://localhost:8085`
4. 注意：使用此方法时，聚合搜索功能可能会受到跨域限制

### 方法三：直接打开

1. 直接双击 `index.html` 文件
2. 在浏览器中打开（某些功能可能受限，如聚合搜索）

## 使用说明

### 灵动岛使用

- **点击灵动岛**：展开/收起灵动岛，查看更多功能
- **搜索音乐**：在展开的灵动岛中输入歌曲名称，按回车或点击搜索按钮
- **播放音乐**：点击搜索结果中的歌曲，开始播放
- **控制播放**：在展开的灵动岛中使用播放/暂停按钮
- **收藏歌曲**：点击收藏按钮将歌曲添加到收藏列表，方便后续查看

### Live2D 模型使用

- **点击模型**：与模型交互，触发动画和表情变化
- **切换模型**：在页面中选择不同的模型，体验不同角色

### 音乐图标使用

- **拖动图标**：点击并拖动音乐图标到任意位置，自定义界面布局
- **点击图标**：显示/隐藏歌词窗口，查看歌曲歌词

### 聚合搜索使用

- **输入关键词**：在搜索框中输入电影或电视剧名称
- **点击搜索**：触发聚合搜索，从多个数据源获取结果
- **横向滚动**：使用鼠标滚轮或拖动滚动条查看更多结果
- **查看详情**：点击电影卡片，查看详细信息和剧集列表
- **播放视频**：在剧集列表中选择集数，开始播放

## 自定义配置

### 灵动岛配置

在 `JS/dynamic-island.js` 文件中，你可以调整以下参数，自定义灵动岛的外观和行为：

- `initialWidth`：初始宽度，默认值为 120
- `initialHeight`：初始高度，默认值为 36
- `expandedWidth`：展开宽度，默认值为 320
- `expandedHeight`：展开高度，默认值为 480
- `animationDuration`：动画持续时间，默认值为 0.3 秒

### Live2D 模型配置

在 `config/model/` 目录中，你可以：

- **添加新模型**：将新的 Live2D 模型文件夹添加到该目录
- **修改配置**：编辑模型的 `config.json` 文件，调整模型参数
- **调整动画**：修改模型的 `motions` 目录中的动画文件
- **自定义表情**：编辑模型的 `expressions` 目录中的表情文件

### 聚合搜索配置

在 `JS/config.js` 文件中，你可以：

- **添加 API 源**：在 `API_SITES` 对象中添加新的数据源
- **调整超时时间**：修改 `AGGREGATED_SEARCH_CONFIG.timeout` 参数
- **配置并行请求**：调整 `AGGREGATED_SEARCH_CONFIG.parallelRequests` 参数
